# MyFamilyHub

MyFamilyHub is a web-based family management system developed using Laravel. The system helps a family preserve family history, manage family member records, organize events, and provide controlled access based on user roles.

## Project Overview

The system was developed for the Tawakkal family as a centralized digital platform to replace scattered manual records and improve access to family information.

## Main Features

- Multi-role login system
- Role-based dashboards
- Family member management
- Search and edit member records
- Event planning and event details
- Photo upload for events
- Family background page
- External Ancestry link for family tree access
- Access control using Laravel middleware

## User Roles

- System Admin
- Family Admin
- Family Member

## Tech Stack

- Laravel
- PHP
- MySQL
- XAMPP
- Tailwind CSS
- Blade Templates
- Eloquent ORM
- Git / GitHub
